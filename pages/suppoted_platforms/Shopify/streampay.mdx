# Shopify Stream**Pay**™ Plugin

The Shopify Stream**Pay** Plugin is a powerful extension for Shopify e-commerce stores, enabling seamless integration with the Stream**Pay** payment provider. With this plugin, merchants can effortlessly process payments through Stream**Pay**™ on their Shopify-based online stores, offering a streamlined and secure payment experience for customers.

## Features

- **Easy Integration**: The Stream**Pay** Plugin seamlessly integrates with your Shopify e-commerce store, allowing you to start accepting payments through Stream**Pay** with minimal setup.

- **Secure Transactions**: Utilize Stream**Pay**'s robust security features to ensure that all payment transactions are secure and protected against fraud.

- **Multiple Payment Methods**: Stream**Pay** supports various payment methods, including credit/debit cards, digital wallets, and more. With this plugin, your customers can choose their preferred way to pay.

- **Real-time Payment Updates**: Stay informed about payment statuses in real time. The plugin automatically updates order statuses when payments are successfully processed.

- **Customizable Checkout**: Tailor the checkout experience to match your store's branding and design. Create a consistent and immersive shopping experience for your customers.

- **Streamlined Refunds**: Process refunds effortlessly through the Shopify admin panel. Refund requests trigger corresponding actions within Stream**Pay** to ensure accurate transaction reversals.

- **Detailed Reporting**: Gain insights into your store's financial performance with detailed transaction reports from Stream**Pay**. Analyze sales data, revenue, and more to make informed business decisions.

## Installation

1. **Clone Repository**: Clone the Stream**Pay** Plugin repository from the Shopify Monorepo.

   ```bash
   git clone https://github.com/stream-protocol/streampay-shopify-monorepo.git
   ```

2. **Navigate to Plugin Directory**: Navigate to the Stream**Pay** Plugin directory within the monorepo.

   ```bash
   cd streampay-shopify-monorepo/plugins/streampay
   ```

3. **Install Dependencies**: Install the required dependencies for the plugin.

   ```bash
   npm install
   ```

4. **Configure Plugin**: Open the `config.js` file and provide your Stream**Pay** API credentials.

   ```javascript
   module.exports = {
     apiKey: 'YOUR_STREAMPAY_API_KEY',
     apiSecret: 'YOUR_STREAMPAY_API_SECRET',
     // Add more configuration options here as needed
   };
   ```

5. **Build Plugin**: Build the plugin to generate the necessary files for integration.

   ```bash
   npm run build
   ```

6. **Integrate with Shopify**: Integrate the built plugin with your Shopify e-commerce store. Follow the Stream**Pay™** Shopify documentation to learn how to enable and configure plugins.

7. **Test Transactions**: Create test transactions using different payment methods to ensure that the integration is functioning correctly.
