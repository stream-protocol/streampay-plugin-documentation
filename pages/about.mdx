# About

The Stream**Pay™** Plugin is a powerful extension designed for e-commerce platforms, enabling seamless integration with the StreamPay™ payment provider. This integration allows merchants to process payments through StreamPay™, providing a secure and user-friendly payment experience for customers.

### Supported E-Commerce Platforms

The StreamPay Plugin currently supports the following e-commerce platforms:**

**1. Medusajs**
**2. Shopify**
**3. WooCommerce**
