## How to Contribute

Contributions to the Stream**Pay**™ Gateway Plugins are welcome and encouraged. Whether you're a developer, designer, or enthusiast, you can make a difference by contributing in various ways:

- **Code Contributions**: If you're a developer, you can contribute by submitting code changes, enhancements, or bug fixes through pull requests.

- **Design and UX**: Designers can contribute by suggesting improvements to the user interface and overall user experience.

- **Bug Reports**: If you come across any bugs or issues, please report them on our [GitHub repository](https://github.com/stream-protocol/ecommerce-payment-plugins-monorepo.git/issues).

- **Feature Requests**: Have an idea to enhance the plugins? Share your thoughts and ideas on our [GitHub repository](https://github.com/stream-protocol/ecommerce-payment-plugins-monorepo.git/issues).

- **Documentation**: Clear and well-maintained documentation is crucial. You can help by improving the existing documentation or adding new sections to provide clarity.

If you're interested in contributing, please review our [contribution guidelines](https://github.com/stream-protocol/ecommerce-payment-plugins-monorepo.git/blob/main/CONTRIBUTING.md) for more details.
